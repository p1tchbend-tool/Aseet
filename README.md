# Aseet

Aseet（**As**sist for spreadsh**eet**）は、Excelスプレッドシートの操作を支援するCLIツールです。  
以下に使い方を記載します。

## セットアップ

[aseet.exeをダウンロード](https://github.com/p1tchbend-tool/Aseet/releases) して、任意のフォルダに配置します。  
配置したexeにパスを通します。

## アンインストール

配置したexeを削除します。  
また以下の一時ファイル保存用フォルダも、必要に応じて削除します。

```text
%LOCALAPPDATA%\aseet
```

## 対応フォーマット

Aseetは以下のフォーマットに対応しています。

- xlam
- xlsm
- xlsx
- xltm
- xltx

> [!NOTE]
> xls（Excel97-2003ブック）には対応していません。

## コマンド

各コマンドについて記載します。

### cat

使用方法: aseet cat [file]  
  
Excelファイルのシート名やセルの値を出力します。  
オプションなしで実行した場合、シート一覧を出力します。  

| オプション    | 説明                                               |
| ------------- | -------------------------------------------------- |
| -a, --all     | 全てのシートの値をページャーで表示します。         |
| -f, --formula | セルの値が数式の場合、値ではなく数式を表示します。 |
| -n, --name    | 指定したシートの値を表示します。                   |

> [!TIP]
> ページャーの操作方法は画面下部に表示されます。  
> なお、マウスで操作することもできます。

### clear

使用方法: aseet clear  
  
Aseetが使用する一時ファイルをすべて削除します。

### diff

使用方法: aseet diff [file1] [file2]  
  
2つのExcelファイルのシート名やセルの内容を比較し、差分を出力します。

| オプション    | 説明                                                                      |
| ------------- | ------------------------------------------------------------------------- |
| -f, --formula | セルの値が数式の場合、値ではなく数式を比較します。                        |
| -o, --open    | 2つのファイルを一時ファイルにコピーし、既定のアプリケーションで開きます。 |

### grep

使用方法: aseet grep [pattern] [file or directory]  
  
Excelファイルまたはディレクトリから、指定した文字列を含むセルを正規表現で検索します。

| オプション        | 説明                                               |
| ----------------- | -------------------------------------------------- |
| -f, --formula     | セルの値が数式の場合、値ではなく数式を検索します。 |
| -i, --ignore-case | 大文字小文字を区別しません。                       |
| -r, --recursive   | サブディレクトリも再帰的に検索します。             |

### sd

使用方法: aseet sd [search] [replace] [file or directory]  
  
Excelファイルまたはディレクトリから、指定した文字列を含むセルを正規表現で置換します。  
セルの値が数式の場合、数式を置換します。

| オプション        | 説明                                   |
| ----------------- | -------------------------------------- |
| -i, --ignore-case | 大文字小文字を区別しません。           |
| -n, --name        | 指定したシートの値を置換します。       |
| -r, --recursive   | サブディレクトリも再帰的に置換します。 |
